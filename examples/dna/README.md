# DNA Example

## Description

Simple DNA-only simulation using the Nucleosome Interaction Coarse-Grained (NICG) model. This example demonstrates basic DNA dynamics with 50 DNA beads representing 525 base pairs in a periodic box.

## Files

- `input.txt`: Chain composition (50 DNA beads)
- `chains.txt`: List of chains, pointing to input file
- `in.bond_settings`: Bond parameters
- `in.run`: LAMMPS simulation script
- `run.sh`: Execution script
- `data.dna`: LAMMPS data file, generated by chromatin builder
- `dna.pdb`: 3D structure, generated by chromatin builder

## Simulation Details

- **System**: 50 DNA beads (525 bp)
- **Box size**: 800 × 800 × 800 Å³
- **Temperature**: 300 K
- **Salt concentration**: 150 mM
- **Timestep**: 10 fs
- **Ensemble**: NVT (Langevin thermostat)

## How to Run

```bash
# Build system and run simulation
./run.sh
```

Or step by step:

```bash
# Build DNA chain
nicg_chromatin_builder.py -f chains.txt -o dna.pdb -d data.dna -b in.bond_settings -hbox 400.0

# Run simulation
lmp < in.run
```
