# 12-Nucleosome Fiber (NRL 189bp) Example

## Description

Regular chromatin fiber simulation with 12 nucleosomes arranged at a nucleosome repeat length (NRL) of 189 base pairs. This example studies local chromatin organization and the effects of uniform nucleosome spacing on fiber compaction.

## Files

- `input.txt`: Chain composition (12 nucleosomes with uniform linkers)
- `chains.txt`: Points to input file
- `in.bond_settings`: Bond parameters
- `in.run`: LAMMPS simulation script
- `run.sh`: Execution script
- `data.chromatin`: LAMMPS data file, generated by chromatin builder
- `chromatin.pdb`: 3D structure, generated by chromatin builder

## Simulation Details

- **System**: 12 nucleosomes with uniform linker lengths
- **Nucleosome Repeat Length**: 189 bp (147 bp nucleosomal + 42 bp linker)
- **Total DNA length**: 2,268 bp
- **Box size**: 800 × 800 × 800 Å³
- **Temperature**: 300 K
- **Salt concentration**: 150 mM
- **Linker DNA**: 4 beads (42 bp) between nucleosomes
- **Ensemble**: NVT (Langevin thermostat)
- **Timestep**: 10 fs

## How to Run

```bash
# Build system and run simulation
./run.sh
```

Or step by step:

```bash
# Build chromatin fiber
nicg_chromatin_builder.py -f chains.txt -o chromatin.pdb -d data.chromatin -b in.bond_settings -hbox 400.0

# Run simulation
lmp < in.run
```
